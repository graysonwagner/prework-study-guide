# Prework Study Guide Webpage

## Description

This study guide was created to help students through the pre work section of the coding bootcamp using html, css, git, and java. 

This is the ultimate key to success!


## Installation

n/a

## Usage

To use the prework study guide you can view notes for each section via the webpage. For suggestions on what to study first use command/option/I on mac or control/shift/I on windows. 

Open the console pannel and inside will be a list of topics learned and what to study first.
